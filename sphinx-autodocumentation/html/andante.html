update([E, ]**F) → None. Update D from dict/iterable E and F.¶

If E is present and has a .keys() method, then does: for k in E: D[k] = E[k] If E is present and lacks a .keys() method, then does: for k, v in E: D[k] = v In either case, this is followed by: for k in F: D[k] = F[k]

andante.grammar_tree_visitor module¶

Defines how one converts a grammar to a Python object.

The grammar defined in the module andante/grammar.py converts input text into a grammar tree. Such a tree is then processed by the grammar-tree-visitor into Python objects.
